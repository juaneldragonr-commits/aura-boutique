# Aura Boutique 💎

Aura Boutique is a premium e-commerce platform built with a focus on visual elegance and robust user experience. This project blends a sophisticated color palette (emerald, stone, and white) with a scalable, modular software architecture.

## 🌟 Key Features

* **Premium Aesthetic:** Interface design focused on cleanliness and minimalism, using a curated color palette to convey exclusivity.
* **Persistent Cart:** Robust implementation of global state management (React Context) synchronized with `localStorage`, ensuring an uninterrupted shopping experience.
* **User Profile Management:** Interactive profile settings allowing users to view and edit personal information in real-time.
* **Modular Architecture:** Highly reusable component structure (Navbar, Footer, AddToCartButton) that facilitates maintenance and scalability.
* **Optimized Navigation:** Streamlined search functionality directed to product collections for a focused user journey.
* **Responsive Design:** Adaptive interface built with Tailwind CSS, offering a fluid experience on both mobile devices and desktops.
* **Next.js Optimization:** Leveraging the App Router for fast and efficient navigation.

## 🛠 Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **State Management:** React Context API + Custom Hooks
* **Icons:** [Lucide React](https://lucide.dev/)
* **Storage:** Browser LocalStorage
* **Deployment:** Vercel
